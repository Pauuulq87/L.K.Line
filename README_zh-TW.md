### <img src="https://api.iconify.design/lucide/triangle-alert.svg?color=%23ff6b6b" width="18" height="18" /> 痛點

傳統八字命理都是文字報告，難以視覺化。無法一眼看出人生軌跡的起起伏伏。

### <img src="https://api.iconify.design/lucide/sparkles.svg?color=%23ffd700" width="18" height="18" /> 解決方案

人生 K 線將命理分析轉化為視覺圖表：

- **蠟燭圖視覺化** — 將人生運勢（1-100 歲）呈現為股票 K 線圖
- **AI 分析** — 性格、事業、財富、婚姻、健康洞察
- **發展風水** — 地理方位與風水建議
- **幣圈運勢**（特供）— Web3 用戶的交易運勢預測
- **免 API** — 複製提示詞到任何 AI（ChatGPT、Claude、Gemini）

### <img src="https://api.iconify.design/lucide/download.svg?color=%23ffd700" width="18" height="18" /> 安裝

**部署到 Vercel**
[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/Pauuulq87/L.K.Line)

**本地設定**
```bash
git clone https://github.com/Pauuulq87/L.K.Line.git
cd L.K.Line

npm install
npm run dev
```

### <img src="https://api.iconify.design/lucide/rocket.svg?color=%23ffd700" width="18" height="18" /> 使用方式

**步驟 1：輸入八字資訊**
- 輸入四柱干支
- 加入大運週期

**步驟 2：複製提示詞**
- 點擊「複製提示詞」按鈕
- 貼到任何 AI 模型（ChatGPT、Claude、Gemini 等）

**步驟 3：匯入結果**
- AI 回傳 JSON 數據
- 貼回 app

**步驟 4：查看 K 線**
- 看到你的人生 K 線圖
- 閱讀詳細分析報告

### <img src="https://api.iconify.design/lucide/chart-candlestick.svg?color=%23ffd700" width="18" height="18" /> 功能特色

| 功能 | 說明 |
|------|------|
| **K 線視覺化** | 1-100 歲運勢顯示為蠟燭圖 |
| **多維度分析** | 事業、財富、婚姻、健康、六親 |
| **發展風水** | 地理方位建議 |
