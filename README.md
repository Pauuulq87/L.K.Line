### <img src="https://api.iconify.design/lucide/triangle-alert.svg?color=%23ff6b6b" width="18" height="18" /> The Problem

Traditional BaZi (八字) fortune telling is text-heavy and hard to visualize. You can't see the ups and downs of your life trajectory at a glance.

### <img src="https://api.iconify.design/lucide/sparkles.svg?color=%23ffd700" width="18" height="18" /> The Solution

Life K-Line transforms fortune analysis into visual charts:

- **Candlestick Visualization** — See your life fortune (1-100 years old) as stock K-line chart
- **AI Analysis** — Personality, career, wealth, marriage, health insights
- **Feng Shui Directions** — Geographic and directional recommendations
- **Crypto Fortune** (Special) — Trading luck predictions for Web3 users
- **No API Key** — Copy prompts to any AI (ChatGPT, Claude, Gemini)

### <img src="https://api.iconify.design/lucide/download.svg?color=%23ffd700" width="18" height="18" /> Installation

**Deploy on Vercel**
[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/Pauuulq87/L.K.Line)

**Local Setup**
```bash
git clone https://github.com/Pauuulq87/L.K.Line.git
cd L.K.Line

npm install
npm run dev
```

### <img src="https://api.iconify.design/lucide/rocket.svg?color=%23ffd700" width="18" height="18" /> Usage

**Step 1: Enter BaZi Info**
- Input your Four Pillars (四柱干支)
- Add Dayun (大運) periods

**Step 2: Copy Prompt**
- Click "Copy Prompt" button
- Paste into any AI model (ChatGPT, Claude, Gemini, etc.)

**Step 3: Import Results**
- AI returns JSON data
- Paste back into the app

**Step 4: View K-Line**
- See your life K-line chart
- Read detailed analysis reports
