### <img src="https://api.iconify.design/lucide/triangle-alert.svg?color=%23ff6b6b" width="18" height="18" /> 痛点

传统八字命理都是文字报告，难以可视化。无法一眼看出人生轨迹的起起伏伏。

### <img src="https://api.iconify.design/lucide/sparkles.svg?color=%23ffd700" width="18" height="18" /> 解决方案

人生 K 线将命理分析转化为视觉图表：

- **蜡烛图可视化** — 将人生运势（1-100 岁）呈现为股票 K 线图
- **AI 分析** — 性格、事业、财富、婚姻、健康洞察
- **发展风水** — 地理方位与风水建议
- **币圈运势**（特供）— Web3 用户的交易运势预测
- **免 API** — 复制提示词到任何 AI（ChatGPT、Claude、Gemini）

### <img src="https://api.iconify.design/lucide/download.svg?color=%23ffd700" width="18" height="18" /> 安装

**部署到 Vercel**
[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/Pauuulq87/L.K.Line)

**本地配置**
```bash
git clone https://github.com/Pauuulq87/L.K.Line.git
cd L.K.Line

npm install
npm run dev
```

### <img src="https://api.iconify.design/lucide/rocket.svg?color=%23ffd700" width="18" height="18" /> 使用方式

**步骤 1：输入八字信息**
- 输入四柱干支
- 加入大运周期

**步骤 2：复制提示词**
- 点击「复制提示词」按钮
- 粘贴到任何 AI 模型（ChatGPT、Claude、Gemini 等）

**步骤 3：导入结果**
- AI 返回 JSON 数据
- 粘贴回 app

**步骤 4：查看 K 线**
- 看到你的人生 K 线图
- 阅读详细分析报告

### <img src="https://api.iconify.design/lucide/chart-candlestick.svg?color=%23ffd700" width="18" height="18" /> 功能特色

| 功能 | 说明 |
|------|------|
| **K 线可视化** | 1-100 岁运势显示为蜡烛图 |
| **多维度分析** | 事业、财富、婚姻、健康、六亲 |
| **发展风水** | 地理方位建议 |
| **币圈运势模块** | 加密货币交易者的运势预测 |
| **免 API** | 通过复制粘贴与任何 AI 模型配合 |
